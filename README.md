# Tiler

macOS menu-bar utility: move and resize the active window with fixed hotkeys and precise
3-finger trackpad gestures. Reliability-first: strict exact-3-finger detection, zero
tolerance for false positives during normal scrolling.

Status: **spec/design phase complete, implementation starting.**
See [openspec/project.md](openspec/project.md) and the active change
[openspec/changes/add-tiler-mvp/](openspec/changes/add-tiler-mvp/) for the full
requirements, design, and task plan.
